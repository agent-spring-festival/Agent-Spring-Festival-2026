### 01 开场歌曲 / Opening Song — 马年春节主题 / Year of the Horse + Spring Festival

### 中文简介
这是“Agent 春晚 2026”的示范节目：提供一份**对各家模型通用**的“开场歌曲生成 Prompt”，用同一套输入让不同模型生成歌词与舞台方案，便于公平对比与复现。

### English
This is a reference show for Agent Spring Festival 2026. It provides a **model-agnostic** prompt to generate an opening song (lyrics + stage plan) so different LLMs can be compared fairly and reproduced.

---

### 你会得到什么 / What you get
- **必出 / Required outputs**: 歌词（含段落结构）/ lyrics (with structure), 舞台方案 / stage plan, 和声/和弦建议 / chord suggestions
- **可选 / Optional**: 英文副歌或双语副歌 / bilingual hook, call-and-response 互动点 / interaction beats

---

### 如何使用 / How to use
1) 打开 `PROMPT.md`，复制里面的 **User Prompt** 到你想用的模型（ChatGPT / Claude / Gemini / 本地模型等）。
2) 把 Prompt 顶部的参数（风格、时长、语言比例等）按需改掉。
3) 保存模型输出为 `output/<model-name>.md`（可选，但推荐）。

---

### 验收标准 / Acceptance checklist
- **春节元素**：至少包含 6 个春节相关意象（例如：年夜饭、灯笼、春联、鞭炮、拜年、红包、团圆、烟花等）
- **马年元素**：至少 2 处与“马/奔腾/骏马/午马/千里”等意象相关的表达（不得过度重复）
- **结构完整**：包含“开场口播/引子 + 主歌 + 副歌 + 过门/桥段 + 结尾收束”
- **可演性**：歌词可唱、节奏点清晰，舞台指令不空泛
- **原创与合规**：不要求“像某某歌手/某某曲子”，不引用受版权保护的明确旋律或歌词
